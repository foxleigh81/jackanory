# Dots

A navigation component that displays dots representing pages. Each dot can be clicked to navigate to that page.

## Usage

```tsx
import Dots from '@/components/navigation/dots';

<Dots
  count={5}
  currentPage={1}
  onChange={(page) => console.log(page)}
  showTooltips
/>;
```
